# Cafe Next.js (Starter)

This is a fully working Next.js starter project for a cafe that mainly serves coffee.
Features:
- Responsive UI / UX (mobile-first)
- Pages: Home, Menu, About, Contact, Cart, Checkout
- Simple Cart state (localStorage)
- Product list with dynamic product pages
- Client-side contact form validation (no backend)

How to run:
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

Files to look at:
- `pages/` - Next.js pages
- `components/` - shared components (Header, Footer, ProductCard)
- `data/menu.js` - sample menu data
- `styles/globals.css` - global styles (responsive)
